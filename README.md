## Описание проекта
Данный проект представляет собой полноценный full-stack Machine Learning сервис для автоматического извлечения данных из отсканированных чеков и документов. Сервис принимает фото сканов документов, сохраняет их в базе данных и анализирует его по ключевым особенностям (даты, имена и сумма). Пользователь может просматривать все загруженные документы также, как и в стандартном File Explorer операционных систем. Помимо фильтров по имени файлов и датам создания возможна сортировка по метаданным загруженных файлов.


## Зависимости
Для запуска системы необходимо наличие Docker. Данный вариант сервиса также предполагает наличие GPU.


## Запуск проекта
1. Склонировать репозиторий
  ```bash
    git clone --recurse-submodules https://gitflic.mtuci.ru/project/i-v-ablyazov-edu-mtuci-ru/piis.git
  ```

2. Создать .env файл в директории FSP с переменными:
  ```bash
    POSTGRES_DB
    POSTGRES_USER
    POSTGRES_HOST
    POSTGRES_PASSWORD
    HASH_SECRET_KEY
    HASH_ALGORITHM
    REDIS_URL
  ```

3. Запустить сервис
  ```bash
    docker-compose up --build
  ```


## Инструкция пользования
1. Запускаем сервис
2. Переходим на http://localhost:3000
3. Регистрируемся и авторизовываемся
4. Переходим во вкладку "Files"
5. Нажимаем "Upload" и выбираем файл
6. После обработки файл автоматически добавится в систему

## Просмотр логов
Для просмотра логов отдельных сервисов необходимо прописать команду:
  ```bash
    docker compose logs -f ИМЯ_СЕРВИСА
  ```
Полный список сервесов представлен в docker-compose.yaml

Для просмотров метрик сервиса необходимо перейти по адресу http://localhost:8000/metrics
